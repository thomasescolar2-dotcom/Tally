# Tally — Landing Page Cinématique

Cette landing page Next.js illustre la vision immersive de Tally pour la conformité produit et les Digital Product Passports. L&apos;interface combine animations Framer Motion et effets GSAP pour offrir une expérience cinématographique inspirée de la science-fiction.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router, TypeScript)
- [Tailwind CSS](https://tailwindcss.com/) pour le style
- [Framer Motion](https://www.framer.com/motion/) pour les micro-interactions
- [GSAP](https://greensock.com/gsap/) pour les animations de fond et le curseur
- [next-themes](https://github.com/pacocoursey/next-themes) pour le mode clair/sombre

## Démarrage

```bash
npm install
npm run dev
```

> **Note :** si l&apos;installation échoue à cause des restrictions réseau, installez les dépendances hors ligne ou via un registre privé accessible.

Ensuite, ouvrez [http://localhost:3000](http://localhost:3000) pour découvrir l&apos;expérience.

## Structure des sections

1. **Hero cinématique** avec fond animé, halo lumineux et CTA vibrant.
2. **Problème / Solution** pour contextualiser la complexité réglementaire et la réponse Tally.
3. **Modules de confiance** présentant les fonctionnalités clés.
4. **Démo interactive** avec mockup animé.
5. **Social proof** en mouvement continu.
6. **CTA final** avec double action immédiate.
7. **Footer minimaliste** avec toggle clair/sombre.

## Accessibilité & performance

- Scroll progress bar, curseur personnalisé et transitions fluides.
- Images optimisées (SVG) et animations orchestrées pour limiter l&apos;usage CPU.
- Responsive mobile-first avec Tailwind CSS.
